## **04-2 반복문으로 흐름 제어**

## **1. while 문과 do~while 문**

주어진 조건이 참인 동안 코드 블록을 반복 실행합니다.

- **while 문**: 시작 전 조건을 먼저 검사합니다(진입 제어). 조건이 처음부터 거짓이면 한 번도 실행되지 않습니다.

```jsx
while (조건식) // 조건식이 거짓일 때가지 반복 실행
{
	실행문
} 
while 문 밖
```

```sql
#include <iostream>
using namespace std;

int main() {
	int count = 0;
	
	while (count < 5) {
		cout << "count : " << count << endl;
		count++;
	}
	
	return 0;
}
```

- **do~while 문: 코드 블록을 먼저 실행한 후 마지막에 조건을 검사합니다(종료 제어). 따라서 최소 한 번은 반드시 실행**됩니다.

```jsx
do // 최소 한 번은 무조건 실행
{
	실행문
} while (조건식) // 조건식이 거짓일 때까지 반복 실행
```

```sql
#include <iostream>
using namespace std;

int main() {
	int count = 0;
	
	do {
		cout << "count : " << count << endl;
		count++;
	} while (count < 5)
	
	return 0;
}
```

```sql
// while과 do~while의 차이점
#include <iostream>
using namespace std;

int main()
{
    int i = 0;
    while (i < 0) { // 조건식이 거짓이므로 반복문은 전혀 실행되지 않는다.
        cout << "i is less than 0" << endl;
        i++;
    }

    int j = 0;
    do {
        cout << "j is less than 0" << endl;
        j++;
    } while (j < 0); // 조건식이 거짓이지만 반복문은 1회 실행된다.

    return 0;
}
// i는 0, j는 1
```

## **2. for 문으로 반복하기**

반복 횟수가 정해져 있거나 배열을 순회할 때 가장 많이 사용됩니다.

- **구성 요소**: `for (초기화; 조건식; 증감식)`의 형태를 가집니다.
    1. **초기화**: 반복 시작 시 한 번만 실행.
    2. **조건식**: 매 반복 전 평가 (거짓이면 종료).
    3. **증감식**: 반복 실행 후 변수 업데이트.

```jsx
for (초기화;조건식;증강식) // 조건식이 참이면 반복 실행
{
	// 반복 실행 코드
}
for 문 밖
```

```sql
#include <iostream>
using namespace std;

int main()
{
	for(int count = 0; count < 5; count++)
	{
		cout << " count : " << count << endl;
	}
	return 0;
}
```

- **장점**: 반복에 필요한 모든 요소를 한 줄에 표기하여 간결하고 무한 루프 방지에 유리합니다.
- **break**: 반복 도중 특정 조건에서 즉시 탈출하고 싶을 때 사용합니다.

```jsx
for(초기화; 조건식; 증강식)
{
	break; // 조건식에 상관 없이 for 문 밖으로
}
for 문 밖
```

```sql
#include <iostream>
using namespace std;

int main()
{
	for(int count = 0; count < 5; count++)
	{
		cout << " count : " << count << endl;
		break;
	}
	return 0;
}
```
