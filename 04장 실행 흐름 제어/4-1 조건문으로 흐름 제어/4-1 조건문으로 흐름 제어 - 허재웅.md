## **4-1 조건문으로 흐름 제어**

## **1. if 문으로 분기하기**

`if` 문은 특정 조건이 참(`true`)일 때만 코드 블록을 실행합니다. 조건이 거짓(`false`)이면 해당 블록은 무시됩니다.

- **if 문**: 조건이 참일 때 동작을 수행하도록 합니다.

```jsx
if (조건식) // 조건이 참일 때 실행문 동작
{
	// 실행문
} 
// 조건이 거짓일 때 실행문 무시
```

- **else 문을 포함하는 if 문:** 조건이 참일 때와 거짓일 때 각각 다른 동작을 수행하도록 합니다.

```jsx
if (조건식) // 조건이 참일 때 실행문 동작
{
	// 실행문
} 
else // 조건이 거짓일 때 실행문2 동작
{
	// 실행문2
}
```

- **else if 문을 포함한 if 문**: 여러 개의 조건을 순차적으로 검사하여 다중 분기를 처리합니다.

```jsx
if (조건식1) // 조건이 참일 때 실행문 동작 
{
	// 실행문1
}
else if(조건식2) // 조건식2가 참일 때 실행문2 동작
{
	// 실행문2
} 
else	// 조건이 모두 거짓일 때 실행문3 동작
{
	// 실행문3
}
```

## **2. switch 문으로 분기하기**

`switch` 문은 하나의 변수나 표현식의 값을 평가하여 여러 `case` 중 일치하는 항목으로 직접 이동하여 실행합니다.

- **동작 방식**: 평가된 값과 일치하는 `case` 레이블을 찾아 구문을 실행합니다. 일치하는 값이 없으면 `default`를 실행합니다.

```jsx
// switch를 활용한 조건 분기
switch (표현식)
{
case 상수1:
	실행문1
case 상수2:
	실행문2
...
default:
	실행문
}
switch 문 밖

// 실행 결과
// 상수1인 경우: 실행문1 실행문2 실행문3...
// 상수2인 경우: 실행문2 실행문3 실행문4...
```

```cpp
#include <iostream>
using namespace std;

int main()
{
    int input_number;

    cout << "1 ~ 5 정수 입력: ";

    cin >> input_number;

    switch (input_number)
    {
    case 1:
        cout << "입력한 수는 1 입니다." << endl;
    case 2:
        cout << "입력한 수는 2 입니다." << endl;
    case 3:
        cout << "입력한 수는 3 입니다." << endl;
    case 4:
        cout << "입력한 수는 4 입니다." << endl;
    case 5:
        cout << "입력한 수는 5 입니다." << endl;
    default:
        cout << "입력한 수는 1 ~ 5 범위 밖입니다." << endl;
    }

    return 0;
}

// 1 ~ 5 정수 입력: 1
// 입력한 수는 1 입니다.
// 입력한 수는 2 입니다.
// 입력한 수는 3 입니다.
// 입력한 수는 4 입니다.
// 입력한 수는 5 입니다.
// 입력한 수는 0 ~ 5 범위 밖입니다.
// 1 ~ 5 정수 입력: 5
// 입력한 수는 5 입니다.
// 입력한 수는 0 ~ 5 범위 밖입니다.
// 1 ~ 5 정수 입력: 7
// 입력한 수는 0 ~ 5 범위 밖입니다.

```

- **break 키워드**: 현재 `case`를 종료하고 `switch` 문을 빠져나가게 합니다. 생략하면 다음 `case`들이 연속적으로 실행되는 'fall-through' 현상이 발생합니다.

```jsx
// **break를 포함한 switch를 활용한 조건 분기**
switch (표현식)
{
case 상수1:
	실행문1;
	break;
case 상수2:
	실행문2;
	break;
...
default:
	실행문3
}
switch 문 밖
// 실행 결과
// 상수1인 경우: 실행문1
// 상수2인 경우: 실행문2
```

```sql
#include <iostream>
using namespace std;

int main()
{
    int input_number;

    cout << "1 ~ 5 정수 입력: ";

    cin >> input_number;

    switch (input_number)
    {
    case 1:
        cout << "입력한 수는 1 입니다." << endl;
        cout << "질문 그만해주세요" << endl;
        break;
    case 2:
        cout << "입력한 수는 2 입니다." << endl;
        break;
    case 3:
        cout << "입력한 수는 3 입니다." << endl;
        break;
    case 4:
        cout << "입력한 수는 4 입니다." << endl;
        break;
    case 5:
        cout << "입력한 수는 5 입니다." << endl;
        break;
    default:
        cout << "입력한 수는 1 ~ 5 범위 밖입니다." << endl;
        break;
    }

    return 0;
}

// 1 ~ 5 정수 입력: 1
// 입력한 수는 1 입니다.
// 1 ~ 5 정수 입력: 5
// 입력한 수는 5 입니다.
// 1 ~ 5 정수 입력: 7
// 입력한 수는 0 ~ 5 범위 밖입니다.

```

- **if vs switch**: `switch`는 단일 변수의 여러 값 비교 시 가독성이 좋고, `if`는 복잡한 조건식이나 범위 검사에 유리합니다.
